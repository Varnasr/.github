# OpenStacks for Change

**Open-source tools for development research, evaluation, and program design.**

Built in India. Shared for impact.

---

### The Ecosystem

| Stack | What It Does | Language |
|-------|-------------|----------|
| [InsightStack](https://github.com/Varnasr/InsightStack) | MEL tools, calculators, econometrics (DiD, PSM, IV, RDD) | Stata, Python, R |
| [FieldStack](https://github.com/Varnasr/FieldStack) | Survey tools, R notebooks, automated reporting | R, Quarto |
| [EquityStack](https://github.com/Varnasr/EquityStack) | Impact evaluation, data cleaning, Jupyter workflows | Python |
| [SignalStack](https://github.com/Varnasr/SignalStack) | Research Rundown newsletter archive | Markdown |
| [RootStack](https://github.com/Varnasr/RootStack) | Database schemas and seed data | SQL |
| [BridgeStack](https://github.com/Varnasr/BridgeStack) | FastAPI backend and REST API | Python |
| [ViewStack](https://github.com/Varnasr/ViewStack) | Frontend dashboards and data exploration | React |
| [PolicyStack](https://github.com/Varnasr/PolicyStack) | South Asia policy and scheme tracker | Python |

### Who This Is For

Development practitioners, evaluators, researchers, and data analysts working across health, education, gender equity, climate, and governance -- primarily in India and South Asia.
