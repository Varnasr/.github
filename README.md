# .github

**Default community health files for all [OpenStacks](https://openstacks.dev) repositories.**

This repository provides shared defaults that GitHub automatically applies to any Varnasr repo that doesn't have its own version of these files.

## What's Here

| File | Purpose |
|------|---------|
| `CONTRIBUTING.md` | How to contribute scripts, data, and documentation |
| `CODE_OF_CONDUCT.md` | Community standards for respectful collaboration |
| `SECURITY.md` | How to report security vulnerabilities |
| `FUNDING.yml` | Sponsor button configuration |
| `PULL_REQUEST_TEMPLATE.md` | Default PR checklist |
| `ISSUE_TEMPLATE/bug_report.md` | Bug report template |
| `ISSUE_TEMPLATE/feature_request.md` | Feature request template |
| `ISSUE_TEMPLATE/question.md` | Question template |
| `ISSUE_TEMPLATE/config.yml` | Issue template chooser with discussion links |
| `profile/README.md` | Profile-level README shown on github.com/Varnasr |

## How It Works

GitHub's [default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) feature means these files apply across all repos unless overridden by a repo-specific version.

## Repositories Covered

InsightStack, FieldStack, EquityStack, SignalStack, RootStack, BridgeStack, ViewStack, PolicyStack, and OpenStacks-for-Change.
